# FinTrack Pro — Frontend 🎨

A premium, high-performance financial dashboard built with **React** and **Vite**, featuring a state-of-the-art **Glassmorphic** design system.

## ✨ Design Principles

-   **Edge-to-Edge Layout**: Optimized for full-screen immersion.
-   **Glassmorphism**: Deep blurs, subtle borders, and glowing accents.
-   **Micro-interactions**: Success animations (`✨ Transaction Saved!`), smooth tab switching, and hover effects.
-   **Data Visualization**: Custom CSS-based charts and summary bars.

## 🚀 Getting Started

```bash
cd frontend/vite-project
npm install
npm run dev
```

## 🧩 Components

-   **AI Coach**: Interactive chat interface with **Quick Action Chips** (History, Budget Check, etc.).
-   **Quick Insights**: Real-time summary of top categories and net balance.
-   **Smart Forms**: Transaction entry with automatic category detection and success triggers.
-   **Responsive Navigation**: Tab-based layout optimized for widescreen displays.

## 🛠️ Tech Stack

-   **Core**: React 18
-   **Build Tool**: Vite
-   **Styling**: Pure Vanilla CSS (No heavy frameworks for maximum performance).
-   **Icons**: Phosphor Icons / Emoji-based symbols.

## 📱 Full Screen Optimization
The application is designed to be viewed in **Full Screen** mode for the best visual experience. The layout dynamically scales from 1360px up to 4K resolutions.
